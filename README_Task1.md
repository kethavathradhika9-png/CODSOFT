# CODSOFT Java Development Internship

This repository contains the tasks completed for the CodSoft Java Development Internship.

## Task 1: Number Guessing Game

A simple Java console-based game where the user guesses a randomly generated number between 1 and 100.

### Features
- Random number generation
- User input using Scanner
- Feedback (Too High / Too Low / Correct)
- Limited attempts
- Option to play multiple rounds
- Score tracking

### Technologies Used
- Java
- Random Class
- Scanner Class
